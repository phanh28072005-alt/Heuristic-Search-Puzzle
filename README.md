# 🧩  XẾP TRANH TÍCH HỢP AI VÀ THUẬT TOÁN TÌM KIẾM HEURISTIC

[![Hugging Face Spaces](https://img.shields.io/badge/%F0%9F%A4%97%20Hugging%20Face-Spaces-blue)](https://huggingface.co/spaces/phanh123556/nhapmonAI)

Ứng dụng trực quan hóa trò chơi xếp hình trượt (Sliding Puzzle) lưới $3\times3$ và $4\times4$. Dự án tập trung vào việc đối sánh hiệu năng định lượng và minh họa cây không gian tìm kiếm giữa các giải thuật AI thông dụng.

## 🚀 Trải Nghiệm Trực Tuyến
👉 Bấm vào đây để chơi game ngay: **[Hugging Face Spaces - nhapmonAI](https://huggingface.co/spaces/phanh123556/nhapmonAI)**

## ✨ Tính Năng Chính
* **Tải ảnh tùy chỉnh:** Tự động cắt và chia nhỏ ảnh tải lên thành các ô lưới mảnh ghép.
* **4 Hàm Heuristic $H(n)$:** Misplaced Tiles, Manhattan Distance, Euclidean Distance, và Linear Conflict.
* **Đối sánh thuật toán:** So sánh trực quan số node duyệt và thời gian chạy (ms) giữa **A\***, **IDA\***, và **BFS**.
* **Đồ thị thời gian thực:** Vẽ cây trạng thái tìm kiếm (Search Tree) và biểu đồ cột so sánh hiệu năng.
* **Auto-Play:** Chế độ AI tự động giải và trình diễn trượt các ô trên bàn cờ.
